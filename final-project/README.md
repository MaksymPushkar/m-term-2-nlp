# Класифікація тендерів Prozorro за допомогою класичного NLP

## Опис проєкту

Даний проєкт реалізує класичний NLP-пайплайн (Natural Language Processing) для автоматичної класифікації тендерів із системи Prozorro.

Дані тендерів збираються безпосередньо через публічний API Prozorro, проходять попередню обробку, перетворюються у TF-IDF ознаки та класифікуються за категоріями закупівель за допомогою моделей машинного навчання.

Метою проєкту є автоматичне визначення категорії тендера на основі його текстового опису та супровідної інформації.

---

## Набір даних

### Джерело даних

Дані отримано через публічний API Prozorro:

https://public-api.prozorro.gov.ua/api/2.5/tenders

### Процес збору даних

1. Отримання ідентифікаторів тендерів із відкритого API.
2. Завантаження детальної інформації для кожного тендера.
3. Витяг текстових полів:
   - назва тендера (title);
   - опис (description);
   - описи товарів та послуг;
   - інформація про CPV-класифікацію.
4. Формування міток класів на основі CPV-кодів.
5. Побудова структурованого датасету для навчання моделей.

## Попередня обробка тексту

Було застосовано такі етапи обробки:

- переведення тексту у нижній регістр;
- нормалізація URL-адрес;
- нормалізація email-адрес;
- заміна дат спеціальними токенами;
- заміна грошових сум;
- заміна чисел;
- видалення спеціальних символів;
- нормалізація пробілів.

Використовувані токени:

- `<MONEY>`
- `<DATE>`
- `<URL>`
- `<EMAIL>`
- `<NUM>`
- `<ID>`

---

## Формування ознак

Для представлення тексту використовувалась TF-IDF векторизація.

```python
TfidfVectorizer(
    ngram_range=(1, 2),
    min_df=3,
    max_df=0.95,
    max_features=50000,
    sublinear_tf=True
)
```

Особливості:

- уніграми;
- біграми;
- нормалізація частот;
- обмеження словника до 50000 ознак.

## Моделі машинного навчання

Було протестовано дві моделі:

### Logistic Regression

```python
LogisticRegression(
    max_iter=2000,
    class_weight="balanced"
)
```

### Linear Support Vector Machine (LinearSVC)

```python
LinearSVC(
    class_weight="balanced"
)
```
